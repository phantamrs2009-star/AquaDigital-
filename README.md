# AquaDigital 💧

> **Transformando desperdício em solução tecnológica e sustentável para escolas públicas.**

O **AquaDigital** é um sistema inteligente desenvolvido para reduzir o desperdício de água em ambientes escolares através da automação, monitoramento em tempo real e reaproveitamento hídrico sustentável.

O projeto busca transformar recursos desperdiçados diariamente — como água de pias, bebedouros, chuvas e condensação de aparelhos de ar-condicionado — em uma solução tecnológica acessível, ecológica e eficiente para escolas públicas.

Este repositório contém o **MVP (Minimum Viable Product)** do sistema AquaDigital, incluindo a interface digital de monitoramento, automação de equipamentos e painel informativo do reservatório.

---

## 🚀 Sobre o Projeto

O desperdício de água é um dos problemas mais recorrentes nas instituições de ensino, impactando diretamente o meio ambiente e os custos financeiros das escolas. O AquaDigital foi criado com o objetivo de solucionar esse problema através de um sistema automatizado capaz de captar, filtrar, armazenar e reutilizar água de maneira inteligente.

O sistema utiliza sensores, controles automatizados e indicadores digitais para acompanhar o funcionamento dos reservatórios e equipamentos em tempo real.

---

## ⚙️ Funcionalidades do MVP

### 💧 Monitoramento Inteligente
Acompanhamento do nível da água nos reservatórios através de sensores e indicadores digitais.

### 🔄 Sistema de Filtragem
Controle automatizado da filtragem da água reaproveitada antes do armazenamento.

### 🚰 Controle de Bombeamento
Ativação e desativação das bombas responsáveis pela movimentação da água dentro do sistema.

### 📊 Painel Analítico
Exibição de gráficos e informações sobre:
- Volume de água armazenado;
- Quantidade de água reutilizada;
- Consumo semanal;
- Status dos equipamentos.

### ☀️ Sustentabilidade Energética
Projeto planejado para funcionar utilizando energia solar fotovoltaica como principal fonte de alimentação.

---

## 🛠️ Tecnologias Utilizadas

Este MVP foi desenvolvido utilizando tecnologias web modernas e acessíveis:

- **HTML5** → Estrutura da interface;
- **CSS3** → Estilização responsiva;
- **JavaScript (ES6)** → Automação e lógica do sistema;
- **Chart.js** → Criação dos gráficos informativos;
- **IoT e Sensores** → Monitoramento inteligente;
- **Energia Solar** → Sustentabilidade energética.
